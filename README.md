# Funciones basicas en AppGameKit

En este repositorio se creo un juego con el motor de juegos AppGameKit, la finalidad es conocer las funciones mas basicas del motor.

### agk::LoadSprite()
Esta funcion sirve para cargar un archivo tipo png desde la ruta que se indique.

### agk::GetVirtualWidth() y agk::GetVirtualHeight()
Ambas funciones retornan como valor flotante el ancho y alto de la ventana donde se ejecutara la aplicación.

### agk::SetSpritePositionByOffset()
Con esta funcion se puede actualizar la posición del sprite indicado, toma como eje el centro del sprite.

### agk::GetRawKeyState()
Devuelve un valor si la tecla indicada esta siendo presionada o no.

### agk::SetSpriteFlip()
Voltea horizontal o verticalmente el sprite indicado.

### agk::GetSpriteWidth() y agk::GetSpriteHeight()
Ambas funciones retornan como valor flotante el ancho y alto del sprite indicado.
